(NÃO FOI UTILIZADO NENHUM RECURSO DE IA PARA A CRIAÇÂO DO CÓDIGO, O OBJETIVO DO TRABALHO É UNICAMENTE MEU APRENDIZADO)

Sistema de gerenciamento de locação de veículos desenvolvido em C# como trabalho final da disciplina de POO.

Conceitos Aplicados

Herança: Classes PessoaFisica e Empresa herdam de Cliente; Automovel, Onibus e Motocicleta herdam de Veiculo
Classes Abstratas: Cliente e Veiculo são classes abstratas que não podem ser instanciadas diretamente
Interfaces: Interface ISeguravel implementada por todos os clientes e veículos
Polimorfismo: Listas de Cliente e Veiculo armazenam objetos de tipos diferentes
Encapsulamento: Atributos com modificadores private, public e protected
Agregação: Classe Locacao agrega objetos de Cliente e Veiculo

Funcionalidades

Cadastro de clientes (Pessoa Física e Empresa)
Cadastro de veículos (Automóvel, Ônibus e Motocicleta)
Registro e gerenciamento de locações
Cálculo automático de seguros e valor total da locação
Listagem de clientes, veículos e locações

Tecnologias

C# / .NET
Aplicação de Console
